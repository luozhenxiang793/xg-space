✏️ 示例题目

例题 1：二次函数综合题

已知抛物线 y = x² - 2x - 3，求：(1) 顶点坐标和对称轴；(2) 与 x 轴的交点坐标；(3) 当 x 取何值时，y > 0？
(1) 配方：y = (x-1)² - 4 → 顶点 (1, -4)，对称轴 x = 1
(2) 令 y = 0：x² - 2x - 3 = 0 → (x-3)(x+1) = 0 → 交点 (-1,0) 和 (3,0)
(3) 由图像可知，当 x < -1 或 x > 3 时，y > 0
💡 技巧：先配方找顶点，再因式分解求交点，最后结合图像判断符号区间

例题 2：几何证明 — 全等三角形

在 △ABC 中，AB = AC，D 是 BC 的中点，E 是 AD 上一点。求证：BE = CE。
证明：∵ AB = AC，D 是 BC 中点
∴ AD ⊥ BC（等腰三角形三线合一）
在 Rt△BDE 和 Rt△CDE 中：
BD = CD（D 是中点），DE = DE（公共边），∠BDE = ∠CDE = 90°
∴ △BDE ≅ △CDE（SAS）
∴ BE = CE ✓
💡 技巧：等腰三角形常用"三线合一"性质，全等证明要找准对应边和角
